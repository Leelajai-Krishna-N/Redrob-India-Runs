## Which Families Dominate?

Top 20 is dominated by Search Ranking Engineer (11), Retrieval Engineer (6), Evaluation/Relevance Engineer (1), Applied ML Engineer (1).

Top 100 is dominated by Search Ranking Engineer (43), Recommendation Engineer (20), Retrieval Engineer (17), Applied ML Engineer (10), NLP Engineer (4).

## Which Families Are Missing?

Missing from the committee-labeled Top 100: Vector Search Engineer, Data Engineer, Backend Engineer, Consulting AI Candidate.

## Which Families Appear Overrepresented?

Recommendation Engineer and adjacent Applied ML/NLP profiles are overrepresented if their evidence is recommendation-only, template-like, or weak on retrieval/evaluation. V1 surfaces many candidates with ranking terms, but committee review discounts repeated role descriptions and missing retrieval infrastructure.

## Which Families Appear Underrepresented?

Evaluation/Relevance Engineer and pure Retrieval Engineer profiles are underrepresented relative to the JD's emphasis on evaluation frameworks, hybrid retrieval, index operations, and ranking-quality measurement. Explicit Search Ranking Engineer evidence is strong in the pool, but it is often hidden under broad AI/ML titles.
